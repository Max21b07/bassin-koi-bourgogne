# Bassin Koi Bourgogne

Site éducatif en français pour un bassin extérieur familial d’environ 2,5 m³ en Bourgogne, avec poissons rouges, carpes koï, fontaine, filtration, plantes et calculateurs côté client.

## Contenu

- Explication du mélange poissons rouges / koï
- Cycle de l’azote et équilibre biologique
- Rôle de la fontaine et de l’aération
- Calculateurs de volume, pompe, air, population, nourriture et qualité d’eau
- Conseils saisonniers adaptés au climat bourguignon
- Problèmes fréquents et actions recommandées

## Utilisation

Ouvrir `index.html` dans un navigateur moderne.

## Développement

Les fichiers principaux sont :

- `index.html`
- `styles.css`
- `app.js`

Le site fonctionne sans backend.

## Audio pédagogique

Les capsules audio restent optionnelles et ne font pas partie du déploiement GitHub Pages.

Procédure locale :

1. Générer les audios avec VoxCPM2 à partir de la voix du propriétaire du site.
2. Placer les fichiers `.mp3` dans le dossier `audio/`.
3. Vérifier le site avec les commandes de contrôle habituelles.
4. Commiter puis pousser les changements.
